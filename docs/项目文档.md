# AI古董鉴定系统 - 项目文档

## 项目简介

AI古董鉴定系统是一个基于多模态人工智能技术的智能古董分析和鉴定平台。系统集成了先进的计算机视觉、自然语言处理和向量数据库技术，为用户提供专业、准确、高效的古董鉴定服务。

## 核心特性

### 🎯 多模态AI分析
- **图像识别**: 基于CLIP和InternVL3.5的先进视觉理解
- **文本理解**: 支持自然语言查询和描述分析
- **流式处理**: 实时输出分析结果，提升用户体验

### 🔍 智能检索系统
- **向量数据库**: 基于Milvus的高性能相似度检索
- **RAG增强**: 利用历史数据提升分析准确性
- **优化路由**: 智能选择最佳处理策略

### 🚀 高性能架构
- **GPU加速**: 支持CUDA加速推理
- **并行处理**: 多线程和异步处理能力
- **缓存优化**: 多层缓存提升响应速度

### 🛠️ 易于部署
- **模块化设计**: 组件独立，易于维护和扩展
- **容器化支持**: Docker和Kubernetes部署
- **配置驱动**: 灵活的配置管理

## 技术栈

### 核心框架
- **Python 3.8+**: 主要开发语言
- **PyTorch 2.0+**: 深度学习框架

## 快速开始

### 1. 环境准备

```bash
# 创建Python虚拟环境
conda create -n ai_antique python=3.9
conda activate ai_antique

# 克隆项目
git clone https://github.com/your-repo/ai-antique-system.git
cd ai-antique-system
```

### 2. 安装依赖

```bash
# 安装核心依赖
pip install -r requirements.txt

# 启动Milvus数据库
docker-compose up -d milvus
```

### 3. 运行系统

```bash
# 启动AI核心服务
python start.py

# 在另一个终端启动前端
cd frontend
npm install
npm start
```

### 4. 基本使用

```python
from ai_core.ai_core import AICore

# 初始化系统
ai_core = AICore()

# 分析古董图像
result = ai_core.analyze_antique_image("path/to/antique.jpg")
print(result["analysis_result"]["description"])

# 流式分析
for chunk in ai_core.analyze_antique_image_stream("antique.jpg"):
    print(chunk, end="", flush=True)
```
